# Input
###
<ui-input placeholder="请输入内容"></ui-input>

```html
<ui-input placeholder="请输入内容"></ui-input>
```

| 属性       | 描述           | 类型     | 默认值  |
|------------|----------------|----------|---------|
| placeholder | 输入框占位符   | String   | 无      |
| disabled    | 是否禁用输入框 | Boolean  | false   |
| value       | 输入框的值     | String   | 无      |
